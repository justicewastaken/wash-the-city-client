# Heartbeat — Proactive Tasks

## Morning Brief (8:00 AM CT)
Every morning, check memory files and prepare a brief summary of:
- Active project status and next steps
- Any scheduled tasks or deadlines
- Anything flagged as urgent from previous sessions
Send the summary to me via Telegram.

## End of Day Memory Save (11:00 PM CT)
Before the day ends, review today's full conversation and save all important context, decisions, tasks, and updates to memory/YYYY-MM-DD.md. This is the most important task — never skip this.

## Weekly Review (Sunday 10:00 AM CT)
Compile a weekly summary of:
- What was accomplished
- Open items and blockers
- Upcoming priorities
Save to memory/weekly-review-YYYY-MM-DD.md
